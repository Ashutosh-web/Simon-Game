Simon Game

Press b/w the colors to Start

How To Play!

1. Press the bullseye to start the game.
2. Lets say R(Red),B(Blue),Y(Yellow),G(Green)
3. Let's suppose the system generates a color R.
4. Then user has to press R.
5. After that system generates a color B.
6. Then user has to press R,B.
7. In the similar manner everytime system generates a random color.
8. User has to input all the color sequence from the beginning.
